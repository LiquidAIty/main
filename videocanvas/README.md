# videocanvas

Root-level video canvas reference submodules for LiquidAIty Media Studio.

Purpose:
- keep useful video/editor/template repos close to the codebase
- preserve upstream update ability
- inspect, adapt, and internalize useful patterns later
- do not wire these directly into the app build until audited

Repos:
- remotion-templates: Remotion motion graphics / video templates
- react-video-editor: Remotion-based browser editor reference
- clip-js: simpler browser video editor / Remotion / FFmpeg reference
